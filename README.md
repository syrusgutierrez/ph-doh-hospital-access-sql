# PH DOH Hospital Access Analysis (SQL)

## Project Overview
This project explores the distribution of Department of Health (DOH) hospitals across Philippine regions using SQL.  
The analysis focuses on hospital availability relative to population size and identifies potentially underserved regions.

## Data Sources
- Department of Health (DOH) hospital facility list
- Philippine Statistics Authority (PSA) regional population data

## Tools Used
- PostgreSQL
- SQL (CTEs, joins, aggregations)
